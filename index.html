Connection Troubleshooting

If you're seeing "Failed to fetch" errors when trying to save to Google Sheets, follow these troubleshooting steps to resolve connectivity issues.

Google Script Settings

  • Ensure your Google Apps Script is deployed as a web app
  • Set "Execute as" to "Me" in deployment settings
  • Set "Who has access" to "Anyone, even anonymous"
  • Re-deploy after making any changes

Verify Script URL

  • Confirm the script URL is correct
  • Make sure you're using the latest deployment URL
  • Test the URL directly in a browser
  • Check for typos in the URL

Security Settings

  • Check your browser's security settings
  • Disable any extensions that might block requests
  • Try in incognito mode to rule out extension issues
  • Ensure no firewall is blocking the request

Script Debugging

  • Check your script's execution logs
  • Verify your script handles POST requests
  • Ensure your script returns proper CORS headers
  • Test with a simple GET request first
